# Mi Página Web Personal

Este proyecto es una página web sencilla creada como práctica para el uso de HTML semántico, imágenes, enlaces, listas y formularios.  
Además, incluye control de versiones con Git y está publicada mediante GitHub Pages.

## Contenido del Proyecto
- index.html con estructura semantica
- Navegacion interna mediante enlaces
- Galeria con imagenes
- Formulario de contacto
- Listas de habilidades
- Footer con enlaces externos
